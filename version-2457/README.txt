静态电影网站文件说明

1. 本站已根据 data_2000.txt 实际解析到的 2000 条影片生成：
   - 首页 index.html
   - 分类总览 categories.html
   - 8 个独立分类页
   - 排行榜 ranking.html
   - 搜索筛选 search.html
   - 2000 个影片详情页，位于 movies/ 目录

2. 所有 HTML 页面均已插入百度统计脚本，脚本位于 </body> 前，不会显示为页面文字。

3. 封面与 Hero 轮播按要求引用站点顶级目录下的 1.jpg 到 150.jpg。
   上传素材中未包含 JPG 图片文件，因此压缩包内不额外生成占位图片。
   部署前可把 1.jpg、2.jpg ... 150.jpg 放入网站根目录。

4. 播放器使用 HLS/m3u8 播放源，已从原 JS 素材中提取 20 条 m3u8 地址，详情页点击播放后会初始化 HLS 播放逻辑。

5. 建议用任意静态服务器或静态托管服务部署整个目录。
